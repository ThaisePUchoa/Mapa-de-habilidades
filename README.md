# Projeto: Mapa de Habilidades - IA aplicada ao Direito

Este projeto faz parte do curso da DIO e tem como objetivo identificar habilidades que podem ser monetizadas, com apoio da Inteligência Artificial.

## Estrutura
- **Etapa 1:** Inventário de Habilidades
- **Etapa 2:** Cruzamento com Necessidades do Mercado
- **Etapa 3:** Caminhos de Monetização
