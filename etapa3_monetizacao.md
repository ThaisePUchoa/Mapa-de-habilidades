# Etapa 3: Caminhos de Monetização

## Checklist
1. Oferecer serviços em plataformas (Workana, 99Freelas, Fiverr)
2. Criar conteúdo gratuito para atrair clientes (LinkedIn, Instagram, YouTube)
3. Montar portfólio simples no GitHub
4. Fazer parcerias locais com escritórios e startups jurídicas
5. Dar aulas ou mentorias online

## Como a IA pode ajudar
- Criar posts para redes sociais
- Gerar modelos de contrato e propostas comerciais
- Sugerir nomes para serviços ou marca pessoal
- Criar plano de conteúdo para atrair clientes
